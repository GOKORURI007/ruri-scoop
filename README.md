## Usage

To add this bucket to scoop, run the following command in PowerShell:

```
scoop bucket add ruri https://github.com/gokoruri007/ruri-scoop
```

then

```
scoop install ruri/<app_name>
```

> Some packages, such as those whose original links are no longer available or cannot be downloaded directly, are hosted on Keybase Filesystem (KBFS) at `/keybase/team/ruri-scoop/hosted`
